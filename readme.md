## Shortcuts:

Common:
- q - Remove selected shapes
- Q - Remove all shapes
- r - Reset scale
- a - Horizontal line
- s - Horizontal ray
- d - Rectangle
- f - Parallel Channel
- g - Vertical line
- z - Ray
- x - Trend line
- Backquote - Open symbol switch menu

Reply:
- w - Backward
- e - Forward
- t - Start / Pause

## Build:

```bash
npm i -g parcel-bundler

npm run build
```
